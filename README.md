# Agentic LinkedIn Post Workflow (Skills Pack)

## What this is
A 4-skill agentic workflow that turns messy notes into a polished LinkedIn post:
1) POST BRIEF
2) Hooks + Outline
3) Draft
4) QA/Polish + Hashtags

## How to run (in Codex)
Run the skills in order. For each step:
- Paste the relevant skill instructions (or reference the file, if your Codex setup supports it)
- Paste the input
- Collect the output and pass it into the next skill

### Step 1 — Post Brief
Use: 01_trashTalk_post_brief.md  
Input: Audience, topic, why now, facts, CTA  
Output: POST BRIEF

### Step 2 — Hooks + Outline
Use: 02_hooks_and_outline.md  
Input: POST BRIEF  
Output: 8 hooks + best hook + outline

### Step 3 — Draft
Use: 03_single_post_draft.md  
Input: hook + outline  
Output: single LinkedIn post draft

### Step 4 — QA + Polish + Hashtags
Use: 04_qa_polish_hashtags.md  
Input: draft + CTA link (or none)  
Output: scorecard + final post + hashtags
